Test for Attract Group

To start using this project you need:
1. Create virtual environment
2. $ git clone https://github.com/dasap89/TEST_AG.git
2. Install pip
3. $ pip install -r req.txt
4. $ python manage.py migrate
5. To run script for task 1 you need to use $ python task_1_script.py
6. To run script for task 2 you need to use $ python task_2_script.py
